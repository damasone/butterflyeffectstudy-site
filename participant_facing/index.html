Reading Frame

The layer stays useful only if it stays narrow.

This is not a sleep score, not a single-night explanation layer, and not a diagnostic surface. The point is to summarize repeated night patterns, baseline-relative shifts, and observed links with next-day burden in a reviewed weekly form.

Audience Switcher

One core, three different contracts.

The same weekly interpretation can be read three ways. The content changes with the audience, but the route governance and report provenance stay attached throughout.

Participant contract

  • What it returnsWeek-at-a-glance, dominant domains, shifts vs baseline, observed links, review nights, and limits.
  • What it does not doNo diagnosis, no disease-risk statement, and no single-night certainty language.
  • Current focusStress / next-day burden only.

Professional contract

  • Same payloadThe companion summary is generated from the exact same weekly report payload.
  • Use caseStructured review, supervision, or participant handoff with clearer operational readout.
  • FormatsHTML, print-ready HTML, JSON, and PDF.

Governance contract

  • Route entryOnly explicitly approved routes that also pass route-card audit can enter.
  • Attached safeguardsAudience contract, route audit, and governance snapshot remain inside the output payload.
  • Why this mattersThe human-facing layer stays serious because it inherits the same methodological discipline as the rest of the system.
